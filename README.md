### clone project:

```
git clone https://github.com/patison5/CentenaireBataille.git
```


### go to the folder:

```
cd CentenaireBataille
```


### install dependencies:

```
npm install
```


### start the server:

```
node server.js
```

### start the server in development mode:

```
npm run dev
```
